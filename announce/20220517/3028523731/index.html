# 夢想之都工作室 Vtuber 二次創作規則

第一條　本規則適用於夢想之都工作室 (以下稱本工作室) 旗下所屬 Vtuber 任何受智慧財產權所保護之內容。由本工作室發布，但非本工作室擁有之智慧財產，不適用本規則。

第二條　本工作室認可任何個人、社團運用本工作室旗下 Vtuber 公開發布之角色美術設定、直播片段、影片片段或本工作室釋出許可使用之創作素材進行二次創作。

第三條　二次創作者不得在任何國家或地區主張或申請智慧財產權。

第四條　二次創作若出現以下情事，本工作室有權要求創作者移除全部或部分創作，若造成本工作室損失，得依法要求賠償。
一、 二次創作侵犯本工作室擁有之合法權益或損及本工作室合作、訂約之第三方之權益。
二、 運用二次創作進行商業行為者。
三、 二次創作有危害本工作室及其下 Vtuber 之名譽、形象者。
四、 違反《兒童及少年福利法》及相關子法散佈創作者。
五、 二次創作仿冒或近似本工作室官方周邊或發布內容。

第五條　未經本工作室許可，不得使用本工作室及旗下 Vtuber 之商標。

第六條　本規則自公告日實施。本工作室保留其解釋、變更及廢止之權利。

【Sachiko Ch. 清水紗智子】立繪公開素材
